# VocabuHero

Build vocabulary faster with a focused, mobile-first learning experience.

## Short description

An Android vocabulary trainer that helps you learn and review words with clean UI and deck-based study flows.

## Highlights

- Deck-based learning flow designed for quick daily practice.
- Smooth Android-first UI with modern Gradle setup.
- Local assets pipeline for vocabulary content.
